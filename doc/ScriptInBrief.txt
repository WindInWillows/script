1.Basic Types:
    1.char:2 Bytes
    2.int:4 Bytes
    3.real:4 Bytes
    4.string:x Bytes
    5.bool:1 Bytes
    6.User-defined structure
2.Grammar
    1.Loop Statements:
        1.while(bool type expression) statement
        2.for(def-expression;boolean type expression;expression) statement
        3.do statement while(boolean type expression);
    2.Condition Statement
        1.if(bool type expression) statement
        2.if(bool type expression) else statement
    3.Jump Statement
        1.break(or continue);(in loops)
        2.return expression;(in functions)
    4.Expressions:
        1.variable++
        2.variable--
        3.++variable
        4.--variable
        5.variable += expression(variable type can be string)
        6.variable (=,-=,*=,/=,%=) expression(%= for integer)
        7.cast expression
            (typename) expression
        8.string\array access:varname[expression]...
        9.unary expression:
            1.prefix:-,!,++,--
            2.postfix:++,--
        10.expression (+,-,*,/,%) expression(% for integer)
        11.expression (==,!=,>,>=,<,<=) expression
        12.bool type expression (&&,||) bool type  expression
    5.Variable Declaration
        Basic-Type identifier[,...] (identifier must begin with one of [a-zA-Z_])
        Basic-Type identifier = initial-value[,...](with initial-value)
        Basic-Type [integer constant]... identifier[,...](for array declaration)
    6.Function Defination & Declaration
        1.Defination
            1.Normal Function Defination
                def return-type identifier (para-type para-identifier[,...]){
                    statement
                }
            2.Structure's member function Defination(can use `this' variable in function body)
                def return-type (Structure Name).identifier (para-type para-identifier[,...]){
                    statement
                }
        2.Pre-declaration(if not used,no need to implement it)
            def return-type identifier(para-type para-identifier[,...]);
    7.Constant:
        1.integer:just support decimal integer
        2.string:"character"(character includes escape character \n,\t,\r,\',\",\?,\b,\f,\\)
        3.real:just support decimal real number
        4.char:'character'(character includes escape character \n,\t,\r,\',\",\?,\b,\f,\\)
        5.boolean:true,false
    8.Load Extension Function:
        1.loadfunc<package-name>{
            return-type class-name ( parameters' types);
        }
    9.Import Source File:
        import "file-path";
    10.Structure definition:
        1.Define a struct
            struct name {
            1.Member Declaration
                type id;[...]
            2.Function Declaration
                def return-type identifier(para-type para-identifier[,...]);[...]
            3.Funtion Defination(can use `this' variable in function body)
                def return-type identifier(para-type para-identifier[,...]){
                        statement;
                    }
            }
        2.Structure member access:
            name.member-name
    11.Dynamic array allocation:
        new<Type>(expression);
    12.Array size getter:
        sizeof array-type-expression
    13.Build-in variable:
        _line_:  line number of this source file
        _file_:  file name of this source file
3.Features In Future
    1.Inline function definition:
        def Type name([Type name,...]) = expression;
